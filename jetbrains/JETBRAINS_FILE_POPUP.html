=== IDEA Vim Keybindings Help ===
-- Navigation --
  -- Global --
    Ctrl-i
    Ctrl-o

  -- Inside Buffer --
    -- Error Navigation --
    • [d: Jump to the next error.
    • ]d: Jump to the previous error.
    • Leader+dw (Space + dw): Open the Problems View Tool Window.

  -- Between Buffers

-- Layout --
  • Leader+e (Space + e): Activate the Project Tool Window.
  • Leader+t (Space + t): Activate the Terminal Tool Window.
  • Leader+r (Space + r): Reopen the last closed tab.


-- Change Next Occurrence --
• Leader+j (Space + j): Removed work under cursor and goes into insert mode without overwriting the default register.

-- Deletion & Pasting --
• x (Normal mode): Delete a character without overwriting the default register.
• p (Visual mode): Paste over the selection without clobbering the yank register.

-- Formatting --
• Q: Format the selected text (remaps Ex mode to use 'gq' for formatting).

-- Debugging & Breakpoints --
• Leader+db (Space + db): Start a debug session.
• Leader+b (Space + b): Toggle a breakpoint on the current line.

-- Window & Tab Management --
• Leader+v (Space + v): Move the current tab right (or split vertically).
• Leader+h (Space + h): Move the current tab down.
• Leader+xs or Alt+q: Close the current content (e.g. a split or tab).
• Shift+Esc: Hide the active window.
• Shift+Arrow Keys:
    – Shift+Up: Stretch the split upward.
    – Shift+Down: Stretch the split downward.
    – Shift+Right: Stretch the split to the right.
    – Shift+Left: Stretch the split to the left.
• Alt+[ and Alt+]: Cycle between splitters (next/previous).


-- Fuzzier Plugin Commands --
• Leader+pf (Space + pf): Launch Fuzzier search.
• Leader+mf (Space + mf): Activate Fuzzy Mover.
• Leader+gf (Space + gf): Launch Fuzzier VCS search.

-- Key Handler Settings --
• IDE-handled keys: Ctrl-A, Ctrl-C, Ctrl-F, Ctrl-K, Ctrl-R, Ctrl-S, Ctrl-V, Ctrl-X.
• Vim-handled keys: Ctrl-B, Ctrl-D, Ctrl-I, Ctrl-O, Ctrl-U, Alt+], Alt+[.

=== End of Help ===
